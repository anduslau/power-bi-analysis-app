## 10. Glossary of Terms

*   **Excel Table:** A structured range in Excel that has column headers, filters, and can be referenced by name in formulas. It expands automatically as data is added.
*   **PivotTable:** An interactive Excel tool used to summarize, analyze, explore, and present large amounts of data.
*   **Formula/Function:** A pre-defined calculation in Excel (e.g., `SUM`, `VLOOKUP`, `IF`). Business rules are encoded here.
*   **Cell Reference:** The address of a cell in Excel (e.g., `B10`). Formulas use these to get values from other parts of the workbook.
*   **Data Validation:** An Excel feature that restricts what data can be entered into a cell (e.g., only numbers, only dates, only values from a list).
*   **Conditional Formatting:** A rule-based feature that changes the appearance of cells (color, font, etc.) based on their values.
*   **Parameter:** In this context, a user-input value (often in a dedicated cell) that controls calculations or filters throughout the report.
